# Omnifood - Healthy Food Delivery Website 🍱

Omnifood is a responsive food delivery landing page built using **HTML**, **CSS**, and **JavaScript**.

The website represents a fictional AI-powered food subscription company that helps users eat healthy meals every day.

---

# 🚀 Features

- Responsive Design
- Modern UI/UX
- Smooth Scrolling Navigation
- Sticky Navigation Bar
- Mobile Friendly Layout
- Reusable Components
- CSS Grid & Flexbox
- Animations & Hover Effects
- Pricing Section
- Testimonials Section
- Featured Meals
- Contact / CTA Section

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Flexbox
- CSS Grid
- Responsive Web Design
